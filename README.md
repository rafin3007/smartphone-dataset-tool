# Smartphone Dataset Lab

A Streamlit web app for building labeled smartphone image datasets. Upload photos, assign them to existing or new phone models (2020–2024), and save organized image collections for training computer vision models.

---

## Features

- Upload images and label them by phone model
- Auto-detects brand (Apple, Samsung, Google, Xiaomi, OnePlus) from model name
- Saves images and metadata to a structured smartphone_dataset/ directory
- View live dataset stats per model class

---

## Setup

`ash
pip install -r requirements.txt
